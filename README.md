# ImmunoType
Comparisons of model analysis and accuracy of classifying immune cell types from single-cell RNA sequencing data


# Research Question
Can machine learning accurately classify PBMC immune cell types from single-cell RNA sequencing data, and which genes contribute most to these predictions?

# Secondary Questions
1. Which model performs best for immune cell classification?
2. Does feature selection improve model performance?
3. Which genes are most influential for each immune cell type?
4. Can interpretable machine learning recover known marker genes?
